# Deliverables

## Core deliverables
1. Installable Python package with a working CLI entry point railway sim.
2. Reproducible example configs under configs.
3. Deterministic outputs per run, including results.csv and run.log in the chosen output directory.
4. Basic regression test suite runnable via pytest.

## Documentation deliverables
1. README quickstart commands that reference files that exist in the repo.
2. README documentation for the live terminal monitor usage and troubleshooting.
3. docs/config_schema.md describing the configuration schema.
4. Process docs in docs folder:
   1. PLANNING describes the workflow and quality gates
   2. TASKS tracks work items and status
   3. WORKLOG records changes with verification
   4. DELIVERABLES defines what outputs must remain supported

## Optional deliverables
1. ASCII plots for headless terminals.
2. Live terminal monitor for progress visualization.
3. Optional UI extras if they are supported by the current dependency setup.
